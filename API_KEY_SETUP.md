# AI Cafe Bot - Configuration

## Important: API Key Setup

Before running the chatbot, you need to add your Google Gemini API Key.

### Steps:
1. Get your API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Open `rag_bot_agentic.ipynb`
3. Find the cell with `GOOGLE_API_KEY = "YOUR_API_KEY_HERE"`
4. Replace `YOUR_API_KEY_HERE` with your actual API key
5. **DO NOT commit your API key to Git!**

### Security Note:
Never share or commit your API key. The `.gitignore` file is configured to prevent accidental commits of sensitive data.
